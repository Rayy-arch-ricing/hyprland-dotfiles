Installation

Copy the configuration files to ~/.config.

Copy the wallpaper collection to ~/.themes.

Copy the theme script to ~/.local/bin.

Run Wallust once to generate the initial color cache.

## Important Notes

### Wallpapers

The wallpaper collection must be copied to:

```text
~/.themes
```

The `theme` script expects wallpapers to be located in this directory.

### User-Specific Paths

Some files contain paths that may need to be adjusted for your system:

* `waybar/style.css`
* `fastfetch/config.jsonc`

By default, these files reference resources using my local paths. Replace `rayy` with your own username or update the paths as needed.

### Wallust

After installing the configuration, run Wallust at least once to generate the required cache files:

```bash
wallust run /path/to/your/wallpaper
```

Several components depend on files generated by Wallust, including:

* Waybar
* Kitty
* Cava

### Script Permissions

Make sure the theme script is executable:

```bash
chmod +x ~/.local/bin/theme
```
